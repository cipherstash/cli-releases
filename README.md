# CipherStash CLI releases

| Platform | Architecture                              |
|----------|-------------------------------------------|
| macOS    | M1, Intel                                 |
| Linux    | ARM64 (glibc, musl), x86_64 (glibc, musl) |
| Windows  | x86_64 (GCC, MSVC)                        |
